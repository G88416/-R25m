PRNG Algorithm Information

Linear Congruential Generator (LCG)

The LCG is one of the oldest and simplest PRNG algorithms defined by the recurrence relation:

Xn+1 = (a Ã— Xn + c) mod m

Where:

  • X is the sequence of pseudo-random values
  • a is the multiplier
  • c is the increment
  • m is the modulus

Mersenne Twister

A widely used PRNG with a period of 219937-1. It provides fast generation of high-quality pseudo-random numbers.

Advantages:

  • Very long period
  • Good statistical properties
  • Fast generation

Xorshift

A class of PRNGs using bitwise XOR and shift operations. Very fast but requires careful parameter selection.

Basic Xorshift32 operation:

x ^= x << 13;
x ^= x >> 17;
x ^= x << 5;

PCG (Permuted Congruential Generator)

A family of PRNGs that combines LCG with permutation functions to improve statistical quality.

Features:

  • Excellent statistical properties
  • Small memory footprint
  • Predictable output (if needed)
